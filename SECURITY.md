# Security Policy

## Supported Versions
Security updates are provided for the active `main` branch.

## Reporting a Vulnerability
Please do not open public issues for vulnerabilities.
Report privately to: `security@seo-tools-platform.local`.

Include:
- Affected endpoint/module
- Reproduction steps
- Impact assessment
- Suggested remediation (if available)

## Response Targets
- Initial acknowledgment: within 72 hours
- Triage decision: within 7 days
- Fix and disclosure timeline: case-by-case based on severity
